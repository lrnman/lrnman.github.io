部署Node.js环境-使用NVM安装多版本

NVM（Node version manager）是Node.js的版本管理软件，使用户可以轻松在Node.js各个版本间进行切换。适用于长期做 node 开发的人员或有快速更新node版本、快速切换node版本这一需求的用户。

安装步骤：

1、直接使用git将源码克隆到本地的~/.nvm目录下，并检查最新版本。

2、激活NVM, 保证重启终端也能用nvm命令。

3、列出Node.js的所有版本。
